Hardware needs Software 🤝

A computer system is made of two parts. One you can touch — one you can't. Let's meet them!

🖱️⌨️🖥️

Hardware

The parts you can see, touch and feel — the mouse, keyboard, hard disk and monitor.

needs ➜
📜✨

Software

A set of instructions that tells the computer what to do. Programs and apps are software at work.

Hardware can't work on its own! A mouse is hardware, but it can't move the pointer without software telling it how and where to go. Computers only understand machine language — 0s and 1s. Software is the bridge between you and the machine.

⚙️ System Software

Manages the computer's hardware and controls how hardware and software talk to each other. It runs the machine behind the scenes.

Operating SystemDevice DriverFirmware Utility SoftwareLanguage Translator

📱 Application Software

Made by programmers so you can do specific tasks — writing, drawing, calculating, or making presentations.

Word ProcessorsPresentation SoftwareWeb Browsers

🧩 The 5 Types of System Software

System software is the bridge between the user and the hardware. Here's what each helper really does.

⚙️
The Boss

Operating System

Manages all the computer's operations — it controls processes and memory, runs both hardware and software, and lets you interact with the machine. A computer can't work without one!

Examples
AndroidiOSWindowsUbuntuLinux
🔌
The Translator's Manual

Device Driver

Manages the hardware connected to a computer and acts as a bridge between the hardware and the operating system. Without drivers, hardware can't work properly.

Examples
BIOS DriverUSB DriverDisplay DriverPrinter DriverSound Card DriverROM Driver
⚡
The Built-in Brain

Firmware

A small program built into hardware and permanently installed. It controls and manages the device's functions in the background. Because it's embedded in the device, it's also called embedded software.

Found inside
Mobile phonesOptical drivesScannersPrintersRemote controllers
🧰
The Housekeeper

Utility Software

Maintains and optimises the computer's performance — keeping it running smoothly, organised, and healthy.

Types & examples
🛡️ Antivirus — Avast, McAfee, Norton 📁 File Management 🗜️ Compression — WinZip, WinRAR 🧹 Disk Cleanup — CCleaner
🔡
The Decoder

Language Translator

Converts a high-level programming language into low-level machine language, because computers understand only 0s and 1s. There are three kinds:

🔧 Assembler

Converts assembly language into machine language.

📦 Compiler

Converts the entire program to machine language all at once.

🚶 Interpreter

Translates and runs the program line by line — great for finding errors.

    📋 Quick Recap

    The big ideas from Chapter 1 — all in one place.

    A computer operates with the help of hardware, software, and a user.

    Software is a set of instructions that tells a computer what to do, and splits into two categories: system software and application software.

    System software manages the computer's hardware and controls the interaction between hardware and software.

    System software includes operating systems, device drivers, firmware, utility software, and language translators.

    Computers understand only machine language (0s and 1s); translators convert human code into it.

    Application software — like word processors, presentation software, and web browsers — lets users perform specific everyday tasks.
